Miami University Explorer

Miami University Explorer is a web application build to explore buildings and courses at Miami University. I built this project from scratch with Remix, Tailwind, Prisma, and SQLite.

Stack
Remix (React.js/Node.js)
SQLite
Prisma
Docker, Fly.io

Stack decisions

I knew I wanted to use JSX to template, Tailwind to style, and Prisma for the ORM. I initially explored using Deno for this project, but it lacked Prisma support at the time. I choose to learn more about Remix, “focused on web standards and modern web app UX”.

Lessons

I learned about Newline delimited JSON (ndjson).
